# castles
AI Coding project with my son. Actually turned into a fun game!

This project was created as a fun way to get my son interested in coding. We started by using emojis and getting some basic controls on the screen to work. I eventually animated some powerups of the emoji sprite and added some more mechanics to make the game a bit more fleshed out. This is a demonstration of coding skill since this was heavily coded with the help of AI.
